===============================Bi-GAN==========================

To run Bi-GAN for EHR dataset - 
	Run "biGan/main_ganOrig.ipynb"
	For training model-
		Set train=True

	For Imputation Testing model-
		Set evalImp=True
		Set missingRate=10 or 20 or 30 or 40 or 50

	For Prediction Testing model-
		Set evalPred=True
		Set pred_len=8 or 7 or 6 or 5
